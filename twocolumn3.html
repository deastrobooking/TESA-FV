Web Game Development

Web Game Development is in its infancy in alot of ways but also just starting to take some big steps into the arena of gaming! WebGL opens up a lot of doors for Developers on any platform. Faster Servers and massive GPU's mean that Webgaming may become a powerhouse in the next few yrs. Especially when you consider the interconnectivity these games allow between different platforms and users.

New Compilers make it possible to bind most any existing game from C++ C# etc. to Javascript for use in the web. It truly is an exciting time to be alive.
